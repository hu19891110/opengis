Edit Using Feature Service
Last modified: 2010/8/2
Author: Wu Yongfeng
If there is editable layer in map service, we can edit features using ArcGIS Web API easily.
Visit Services in Browser
Visit Feature Services by URL like "http://localhost:8777/giserver/rest/service/FeatureService", Feature Services depend on Map Services.
Using services in ArcGIS Web API
GIServer has the similar interface like ArcGIS Server REST, so we can use ArcGIS Server Web APIs, Such as Flex, Silverlight, JavaScript.
