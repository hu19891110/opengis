使用地图服务
更新: 2010/8/1
作者: 吴泳锋
GIServer的地图服务提供和ArcGIS Server REST接口基本一致的访问方式， 因此，通过浏览器和ArcGIS Web API都可以非常容易地使用GIServer发布的地图服务。
在浏览器中访问服务
通过类似这样的网址可以在浏览器中访问到GIServer的服务主页面：http://localhost:8777/giserver/rest/service/
使用ArcGIS Web API调用服务
GIServer采用兼容ArcGIS Server REST接口的设计，因此可以在前台使用各种ArcGIS Server的客户端API， 比如 Flex、 Silverlight、 JavaScript。
