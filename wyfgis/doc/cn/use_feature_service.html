使用要素服务进行编辑
更新: 2010/8/2
作者: 吴泳锋
如果使用GIServer发布的地图服务中有可编辑的图层，我们可以使用ArcGIS Web API很方便地进行要素编辑。
在浏览器中访问服务
通过类似这样的网址可以在浏览器中访问到要素服务主页面：http://localhost:8777/giserver/rest/service/FeatureService， 要素服务依赖于地图服务，点击某个服务即可查看服务中的要素图层：
使用ArcGIS Web API调用服务
GIServer采用兼容ArcGIS Server REST接口的设计，因此可以在前台使用各种ArcGIS Server的客户端API， 比如 Flex、 Silverlight、 JavaScript。
